# Iceworks 组件创建插件

使用低代码的方式创建前端组件。

## 用法

![create-component](https://img.alicdn.com/tfs/TB1.WORe5cKOu4jSZKbXXc19XXa-960-600.gif)

1. 输入组件名（将用作文件夹名）
2. 选择区块
3. 点击「生成组件」按钮，生成组件代码
