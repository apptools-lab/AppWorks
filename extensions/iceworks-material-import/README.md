# Iceworks 物料添加插件

使用可视化的方式添加物料到应用中。

## 用法

![使用示例](https://user-images.githubusercontent.com/56879942/87401390-a63a1700-c5ec-11ea-91a6-f7d81ac5fd77.gif)

1. 选择需要插入物料的位置
2. 点击需要使用的区块或者组件，物料代码将自动添加到相应的位置中
