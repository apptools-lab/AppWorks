# Iceworks 物料导入插件

使用低代码的方式导入物料。

## 用法

![物料导入](https://img.alicdn.com/tfs/TB1GD1Wa9R26e4jSZFEXXbwuXXa-1393-892.gif)

1. 选择需要插入物料的位置
2. 点击需要使用的区块或者组件，相应的代码自动添加到代码中
