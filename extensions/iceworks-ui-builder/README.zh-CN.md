使用可视化搭建的方式创建用户界面。

### 激活

通过 `⇧⌘P` ( MacOS ) 或 `Ctrl+Shift+p` ( Windows ) 快捷键唤起命令面板，输入 `Iceworks: 生成组件` ，激活插件。

### 使用

1. 拖拽左侧的组件到中间的画布面板中；
2. 按下 `⌘+S` ( MacOS ) 或 `Ctrl+S` ( Windows ) ，在输入框中填写组件名，按下回车键即可生成组件到项目中。
