### iceworks CLI

使用 iceworks CLI 开发自定义物料集合。

安装工具：

```bash
$ npm i -g iceworks
```

初始化物料集合：

```
$ mkdir materials-example && cd materials-example
$ iceworks init material
```

完整文档请参考[物料开发](https://ice.work/docs/materials/about)。
